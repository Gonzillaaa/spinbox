# Spinbox v0.1.0-beta.7

## Features

Git hooks integration merged to main branch.

**Pre-commit Hook**:
- black (formatting)
- isort (imports)
- flake8 (linting)

**Pre-push Hook**:
- pytest before push

```bash
spinbox create myproject --python           # With hooks
spinbox create myproject --python --no-hooks # Without hooks
```
