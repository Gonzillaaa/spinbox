# Spinbox v0.1.0-beta.6

## New Features

### Git Hooks Integration
Automatic quality gates for Python projects:

**Pre-commit**: black, isort, flake8 on staged files
**Pre-push**: pytest before pushing

```bash
spinbox create myproject --python           # Hooks installed
spinbox create myproject --python --no-hooks # Skip hooks
```
