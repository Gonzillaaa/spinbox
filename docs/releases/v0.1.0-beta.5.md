# Spinbox v0.1.0-beta.5

## New Features
- **Automatic Dependency Management**: `--with-deps` flag for automatic package management
- **TOML-Based Dependencies**: Component dependencies defined in structured templates
- **Setup Scripts**: Automatic generation of dependency installation scripts

## Supported Dependencies

**Python**: fastapi, uvicorn, pydantic, sqlalchemy, asyncpg, redis, celery, chromadb, beanie, motor, openai, anthropic, langchain

**Node.js**: next, react, axios, TypeScript, express, cors, tailwindcss

## Usage
```bash
spinbox create myapi --fastapi --with-deps
spinbox create webapp --fastapi --nextjs --postgresql --with-deps
spinbox add --redis --chroma --with-deps
```

## Testing
- 72+ tests passing
- All component generators tested with `--with-deps`

## Update
```bash
spinbox update
```
